# SQLTerminal 0.1.1

Second Windows release, on the new `0.1.1` marketing line.

## What's new
- **Results filter** — a full-text filter field at the top of the results grid:
  live, case-insensitive, matches any column, with an "N of M" match count. Esc
  clears it.
- **Resizable columns** — result columns can be dragged to any width (the old
  440 px cap is gone), so long values fit.
- **About box** — GitHub and Report Issue links, plus a dedication.
- **UI polish** — cleaner pane splitter (a single hairline seam with tighter
  gutters) over the rounded-card panes.

Auto-updates via WinSparkle (EdDSA-signed appcast). Existing 0.1.0 installs are
offered this update automatically.
